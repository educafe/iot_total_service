BT-HC06-Basic.ino
===================
라즈베리파이에서 rfcomm 명령으로 문자열 데이터를 전송하고, 아두이노에서 수신한 문자열을 출력해 보는 예제로
rfcomm 파일을 이용하여 데이터 전송이 가능함을 시사

BT-HC06-Led_num.ino
======================
라즈베리파이로부터 ASCII '1'을 수신하면 Led를 ON 한 후 라즈베리파이로 다시 'LED ON' 문자열을 전송하여 
라즈베리파이에서 출력하도록 하는 예제.
05.bt_rfcomm-hc06.c 프로그램과 매칭하여 실행

BT-HC06-Led_str
================
라즈베리파이로부터 문자열 'ON'을 수신하면 Led를 ON 한 후 라즈베리파이로 다시 'LED ON' 문자열을 전송하여 
라즈베리파이에서 출력하도록 하도록 하고 'OFF'를 수신하면 LED를 Off 한후 'LED OFF"를 반송하는 예제로 
문자열 수신시 다른 방법을 사용하고 있음을 보기 위한 것.
06.bt_rfcomm-hc06.c 프로그램과 매칭하여 실행
