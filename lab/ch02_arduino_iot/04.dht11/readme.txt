1) 라이브러리 포함
------------------
temp_dht11.ino 파일에서 dht11.h 파일을 포함하기 위해서  
->dht11_mylib.zip 파일을 "스케치->라이브러리포함하기" 메뉴에서 라이브러리 포함해야 한다
-> 해당 라이브러리는 Q:/lab/libraries/ 하위에 저장되며
--> #include <dht11> 과 같은 형식으로 헤더파일을 포함한다

압축 파일을 풀어서 ~.ino 파일과 동일한 디렉토리에 위치하는 경우
#include "dht11.h" 등으로 수정하면 된다

2) 라이브러리 삭제 (원상태로 복귀)
라이브러리가 저장되어 있는 디렉토리 (Q:\lab\libraries)에서 해당 라이브러리 zip 파일을 삭제하고
arduino preference 파일이 존재하는 곳에서 모든 preference 파일을 삭제한다

Q:\lab\libraries\dht11_mylib (legacy)의 라이브러리 dht11_mylib 사용 에러 발생시
=================================================
해당 폴더 하위에는 1개의 라이브러리만 있어야 한다. 즉 "dht11.h" 만 존재하든지 
<dht11.h>를 담고 있는 zip 파일만 존재하든지 해야 한다.
dht11.cpp 파일도 존재해서는 안된다.
